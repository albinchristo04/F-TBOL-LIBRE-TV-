# Match Previews

This directory contains blog posts about upcoming match previews.

Each preview should be a markdown file with the following frontmatter:

```yaml
---
title: "Team A vs Team B Preview"
description: "Preview of the upcoming match between Team A and Team B"
date: 2024-01-01
league: "League Name"
team_a: "Team A"
team_b: "Team B"
match_date: "2024-01-15"
match_time: "20:00"
featured: false
---
```
