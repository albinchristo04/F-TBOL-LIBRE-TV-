# Match Recaps

This directory contains blog posts about match recaps and analysis.

Each recap should be a markdown file with the following frontmatter:

```yaml
---
title: "Team A vs Team B Recap"
description: "Recap of the match between Team A and Team B"
date: 2024-01-16
league: "League Name"
team_a: "Team A"
team_b: "Team B"
score: "2-1"
match_date: "2024-01-15"
featured: false
---
```
